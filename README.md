# PDF and Document Automation System

This project contains tools and services for document automation and processing, including:
- A backend FastAPI service (`doc_automation/backend`) which processes files, edits PDF/DOCX templates, and integrates with AI capabilities.
- A frontend Vite/TypeScript web application (`doc_automation/frontend`) that provides a user-friendly drag-and-drop interface for managing automation pipelines.
- An older or separate utility folder (`pdf_automation`).

## Project Structure
- `doc_automation/backend/`: Backend codebase.
- `doc_automation/frontend/`: Frontend codebase.
- `pdf_automation/`: PDF automation modules.
